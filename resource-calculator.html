How to use this Resource Calculator
Instructions:

1.   To calculate your deliveries, change ONLY the value in Second input which will turn white when you hover, Since they are the lowest value per ratio, they are our limiting factor. of any car

2.   All of the tables will update automatically to the Correct Numbers. No of cars will tell you the proper number of cars needed to make a delivery to any industry.


Lock Button:

Click the lock button to lock the input fields. This will set incriment limits to input such that it will partially help the user in getting full cars.
Note: This is a beta feature and will be optimised in the future.
